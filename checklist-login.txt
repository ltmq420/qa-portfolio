Чек-лист для тестирования формы логина на GitHub

- Логин с правильными данными -> вход выполнен
- Логин с неправильным паролем -> ошибка "Неверный логин или пароль"
- Логин с пустым полем логина -> ошибка "Заполните  это поле(логин)"
- Логин с пустым полем пароля -> ошибка "Заполните это поле(пароль)"
- Логин с несуществующим пользователем -> ошибка "Неверный логин или пароль"
- Кнопка «Continue with Apple» → открывается страница входа Apple
- Кнопка «Continue with Google» → открывается окно выбора аккаунта Google
- Вход через Google с правильным аккаунтом → вход выполнен
- Вход через Google с отменой → возврат на страницу логина
- Вход через Apple с правильным аккаунтом → вход выполнен
- Вход через Apple с отменой → возврат на страницу логина
- Кнопка "Войти" доступна на всех этапах заполнения полей.
