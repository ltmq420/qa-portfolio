Тест-кейс: Проверка входа с неправильным паролем на GitHub

ID: TC-01
Название: Проверка поведения системы при вводе неправильного пароля
Предусловие: Открыта страница логина https://github.com/login
Шаги:
  1. Ввести существующий логин (например, свой или testuser)
  2. Ввести заведомо неправильный пароль
  3. Нажать кнопку «Sign in»
Ожидаемый результат: Появляется сообщение об ошибке "Incorrect username or password."
Фактический результат: Сообщение об ошибке появилось
Статус: Пройден
